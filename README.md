# SCT_ML_2 - Customer Segmentation using K-Means Clustering

## 📌 Project Overview
This project is part of my SkillCraft Technology Machine Learning Internship.  
The goal is to segment mall customers into different groups based on their purchasing behavior using the K-Means Clustering algorithm.

---

## 📊 Dataset Information
The dataset contains customer details such as:
- Customer ID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1-100)

Source: Mall Customers Dataset

---

## 🎯 Objective
To identify distinct customer groups for better marketing strategies and targeted promotions.

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn (KMeans)

---

## 🧠 Machine Learning Technique
- Unsupervised Learning
- K-Means Clustering Algorithm
- Elbow Method to find optimal number of clusters

---

## 🔥 Steps Performed
1. Imported required libraries
2. Loaded dataset
3. Selected features (Annual Income, Spending Score)
4. Used Elbow Method to find optimal clusters
5. Applied K-Means algorithm
6. Visualized customer segments

---

## 📈 Results
Customers were grouped into 5 clusters:
- High income, high spending (Target customers)
- High income, low spending
- Low income, high spending
- Low income, low spending
- Average customers

---

## 📌 Conclusion
K-Means clustering successfully helped in identifying meaningful customer groups that can be used for targeted marketing strategies.
